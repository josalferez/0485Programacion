una cafetera o un bote de forma individual. Sin embargo, en la vida real (y en las aplicaciones profesionales),

Vamos a crear una "Cafetería" que gestiona un almacén de botes utilizando un array de objetos.

El Reto: El Almacén de la Cafetería (Arrays de Objetos)
Contexto: Nuestra cafetería ha crecido y ahora tenemos una estantería donde guardamos varios botes de café. 
Debemos ser capaces de revisar todos los botes para saber cuál necesita reposición.

1. El Tipo Enumerado (en TipoCafe.java)
(Mantenemos el mismo): ARABICA, ROBUSTA, MEZCLA, DESCAFEINADO.

2. La clase Bote (en Bote.java)
Atributos: TipoCafe variedad, int gramos.

Constructor: public Bote(TipoCafe variedad, int gramos).

Método mostrarInfo(): Imprime "[Variedad] - Quedan [gramos]g".

3. La clase de prueba GestionAlmacen (en GestionAlmacen.java)
Aquí es donde aplicaremos el Array de Objetos.

En el main, los alumnos deben:

Crear el Array: Declarar un array capaz de guardar 3 objetos de tipo Bote. Bote[] estanteria = new Bote[3];

Instanciar los objetos dentro del array:

estanteria[0] = new Bote(TipoCafe.ARABICA, 500);

estanteria[1] = new Bote(TipoCafe.ROBUSTA, 150);

estanteria[2] = new Bote(TipoCafe.DESCAFEINADO, 800);

Recorrer el array (Bucle For):

Usar un bucle para llamar al método mostrarInfo() de cada bote.

Lógica de búsqueda (El Reto):

Los alumnos deben crear un bucle que recorra el array y muestre un mensaje de "¡REPOSICIÓN URGENTE!" solo 
para aquellos botes que tengan menos de 200 gramos.
