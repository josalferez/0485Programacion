# 📘 Programación - 1º DAW

Repositorio de la asignatura **Programación** del ciclo formativo de grado superior **Desarrollo de Aplicaciones Web (DAW)**.

---

## 🎯 Objetivos de la asignatura

En este módulo se trabajan los fundamentos de la programación utilizando **Java**, desarrollando habilidades como:

* Pensamiento lógico y algorítmico
* Resolución de problemas
* Programación estructurada y orientada a objetos
* Buenas prácticas de desarrollo

---

## 🧠 Contenidos

A lo largo del curso se abordan los siguientes bloques:

### 🔹 Introducción a la programación

* Algoritmos
* Diagramas de flujo
* Pseudocódigo

### 🔹 Java básico

* Variables y tipos de datos
* Operadores
* Entrada y salida de datos

### 🔹 Estructuras de control

* Condicionales (`if`, `switch`)
* Bucles (`while`, `for`, `do-while`)

### 🔹 Programación modular

* Métodos / funciones
* Paso de parámetros

### 🔹 Programación orientada a objetos (POO)

* Clases y objetos
* Atributos y métodos
* Encapsulación
* Herencia
* Polimorfismo

### 🔹 Colecciones y estructuras de datos

* Arrays
* ArrayList

### 🔹 Acceso a datos (introducción)

* JDBC
* Patrón DAO
